# licensegen

Generate LICENSE files for open source projects. Zero dependencies.

## Usage

```bash
licensegen generate mit --name "Your Name"   # Generate MIT license
licensegen list                               # Show available licenses
licensegen detect                             # Detect existing license type
```

## Available Licenses

MIT, Apache 2.0, GPL 3.0, BSD 2-Clause, ISC, Unlicense

## Requirements

- Python 3.6+ (stdlib only)
